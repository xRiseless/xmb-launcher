# General: 
Sony's PS3 XMB-like launcher for Steam Deck (or no) written with DeepSeek.
<img width="1599" height="987" alt="image" src="https://github.com/user-attachments/assets/8ef4c88e-1e8a-470d-8bd0-64be8d0208d0" />

This is an early alpha. Right now, the launcher features a PS3's XMB-like interface and basic support for parsing/launching Steam games.

# Contributing:
If you want to help in developing - feel free to fork this project and customize it for your needs.

# How to use:
- Install Python 3.13
- Use "pip install -r requirements.txt" to install depencies
- Launch "main.py"
- Use "Update Steam Game List" to parse your games into launcher
- You also can add caterogies\subcategories\games manually using .json files in data folder (syntax is very simple)
